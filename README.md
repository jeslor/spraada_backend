# Spraade 🌐💬🤝

Spraade is a social app backend built with **NestJS**, designed to help people connect, share, and grow with like-minded individuals.

---

## Features

- User authentication and profile management 🔐
- Connect with like-minded people 🤝
- Create and join communities 🌐
- Post and interact with content 💬
- Real-time notifications and messaging 🔔

---

## Tech Stack

- **Backend:** NestJS 🏗️
- **Database:** PostgreSQL / MongoDB (choose based on your preference) 🗄️
- **Authentication:** JWT / Passport.js 🔑
- **API Docs:** Swagger 📝

---
